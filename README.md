# Yes Teacher

Yes Teacher is a web-based attendance management system for tuition centres.

The system is built using React and Vite. It will allow teachers/admins to manage classes, students, teachers, and session-based attendance.

## Main Features

- Class management
- Student management
- Teacher management
- Session-based attendance
- Manual attendance status:
  - Attend
  - Late
  - Absent
  - Not submitted

## Tech Stack

- React
- Vite
- JavaScript / JSX
- CSS
